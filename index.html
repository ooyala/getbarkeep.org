Who is the Barkeep?

The Barkeep is a fast, fun way to review code. Engineering organizations can use it to keep the quality bar high. With it you can watch commits made to any Git repository, see diffs, write comments, and have those comments emailed to your fellow committers. Read how to install it on Github.
